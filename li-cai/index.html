# 主流工具

  • 存银行

  • 股票

  • 基金（比较推荐）：路径： 支付宝 => 基金排行 => 类型 => 概括指标：

    • 不少于5年，
    • 资产5亿以上
    • （A结尾 不收管理费，C结尾 以后可能会收管理费）
    • 任期：投资经理不要经常变来变去
    • 投入类型：
      • 定投（比较推荐）：定时一个月买一次； 什么时候开始不重要，重要是开始 收益多少不重要，重要的是止盈（落袋为安）；
      • 买入
    • 投资策略：年终止盈或者升幅（20%）止盈

  • P2P 高风险，容易血本无归

  • 募资

  • ...等等
